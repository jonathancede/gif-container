# GIFHome - GIF Container Server

Steps to run the server:

- Clone this repository.

```bash
$ git clone https://github.com/jonathancede/gif-container.git
```

- Be connected to a local Mongoose database.

```bash
$ mongod
```

- Execute the command:

```bash
$ npm install
```

- Execute the command:

```bash
$ npm run dev
```
